# Online Shop Project

## Features
- User-friendly interface
- Product catalog with categories
- Shopping cart functionality
- Secure checkout process
- Order history management
- Admin panel for product management
- Payment processing support

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/buffchro-source/ONLINE-SHOP.git
   cd ONLINE-SHOP
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Set up your environment variables in a `.env` file as per the provided `.env.example`.
4. Start the application:
   ```bash
   npm start
   ```

## Usage
- Access the online shop in your web browser at `http://localhost:3000`.
- Register a new account or log in with existing credentials.
- Browse products, add them to your cart, and proceed to checkout.

## Customization Guide
- To customize the look and feel, modify the CSS files located in the `assets/css` directory.
- For adding new features, consider extending the existing classes in the `src` directory.
- Update product details in the admin panel accessible only to admin users.
